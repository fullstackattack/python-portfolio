NASA Space Dashboard API

Python and Flask backend that integrates multiple NASA public API endpoints and displays the data through a simple, styled dashboard. 
This project demonstrates API consumption, backend routing, secure environment variable management, and basic frontend rendering.

Key Features

*Astronomy Picture of the Day (APOD)
  Fetches the daily image + description and displays it in a styled template.

*Near-Earth Objects (NEO) Feed
  Retrieves asteroid data for a selected date and summarizes counts and object details.

*EPIC Earth Images
  Shows the most recent Earth image captured from NASA’s DSCOVR spacecraft.

*NASA Image Search
  Allows keyword-based search of NASA’s historical image archive.

Technologies Used

* Python 3
* Flask
* Requests
* Jinja2 Templates
* HTML/CSS
* NASA Open APIs
* python-dotenv
