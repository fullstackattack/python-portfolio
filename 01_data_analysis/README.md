This folder contains exploratory data analysis projects that showcase Python data-science fundamentals using pandas, NumPy, and Plotly. Each notebook demonstrates importing and cleaning files along with exploration, visualization, and interpretation.

1. Athletes Analysis

Goal: Explore relationships between runners’ heart rate and mile times.
Skills used: pandas grouping, aggregation, pivot tables, bar charts  

- Find each athlete’s minimum and maximum heart rate  
- Count number of runs per athlete  
- Visualize with grouped and horizontal bar charts

File: `athletes_analysis.ipynb`  

2. Student Performance Data
Goal: Investigate factors that impact exam scores.

Skills: filtering, correlation, data cleaning, visualization  
Highlights:
- Clean missing data  
- Visualize score distributions  
- Identify variables correlated with high performance

File: `student_performance_data.ipynb`  
Dataset: `datasets/students.csv`
